# Reach Circle

A lightweight, fully client-side reach visualization mod for Minecraft
1.21.3 and 1.21.5.

Reach Circle displays accurate reach distance indicators in real time
for both you and other players. It improves combat awareness,
positioning, and interaction precision.

## Features

\[+\] Displays your reach distance\
\[+\] Displays other players' reach distance\
\[+\] Dynamic color changes when entering reach\
\[+\] Dynamic color changes when leaving reach\
\[+\] Visual indicator when you enter another player's reach\
\[+\] Fully customizable colors\
\[+\] Clean and minimal rendering\
\[+\] Performance friendly\
\[+\] 100% client-side

## Use Cases

\[\>\] PvP practice\
\[\>\] Combat range awareness\
\[\>\] Positioning improvement\
\[\>\] Visual training tool\
\[\>\] Technical gameplay analysis

## Configuration

The mod includes configurable options that allow you to:

\[\>\] Change circle colors\
\[\>\] Configure enter and exit reach color behavior\
\[\>\] Adjust visuals to match your resource pack or UI setup

## Supported Versions

\[+\] Minecraft 1.21.3\
\[+\] Minecraft 1.21.5\
\[-\] Other versions are not supported

## Installation

1.  Install Fabric loader.\
2.  Download the correct version of Reach Circle or compile it using the source code.\
3.  Place the .jar file into your mods folder.\
4.  Launch the game.
